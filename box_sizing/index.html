box-sizing: border-box

Das normale Verhalten von Browsern ist, dass paddings (oder borders) das Element nach außen vergrößern. Wenn man also mit festen Abmessungen arbeitet (width, height), muss man immer die paddings einrechnen:

width: 100px + padding: 10px
Die Ergebnisbreite ist 120px: width + padding-left + padding-right.

Die CSS Eigenschaft box-sizing: border-box ändert das Verhalten, dass border und padding nun nach "innen" wachsen, also die Gesamtbreite des Elements nicht beeinflussen. Es ist nicht schwer vorzustellen, dass so das Erstellen eines Layouts wesentlich einfacher ist.

Für IE < 8 kann das Feature per polyfill nachgerüstet werden.
