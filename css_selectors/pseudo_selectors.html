Übungen

  1. Erstellen Sie via CSS eine Tabelle mit Zebramuster, deren Zeilen umrandet werden, falls die Maus über der Zeile ist.
  2. Sorgen Sie per CSS dafür, dass Links nur bei Hover unterstrichen werden.
  3. Sorgen sie per CSS dafür, dass vor allen Links automatisch der Text "Link:" eingeblendet wird.
