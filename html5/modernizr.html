CSS Schulung

Modernizr bietet mehrere Funktionen:

  • HTML5-Tags in älteren Browsern
  • Feature-Detection via CSS-Klassen im html-Tag (Elements-Tab in Google Chrome Dev-Tools)
  • Feature-Detection via JavaScript
  • fehlende Features können ignoriert oder oft via Polyfills nachgerüstet werden.

Wichtig: Modernizr muss im head-Tag eingebunden werden!
