CSS Features

Viele dieser Eigenschaften funktionieren erst ab IE9 (die Animationen z.B. sogar erst ab IE10). Viele davon können nachgerüstet werden, z.B. über CSS3PIE.

Ein guter Anlaufpunkt, welche Features man (schon) verwenden kann, ist caniuse.

Recht aktuelle Statistiken, welche Browser noch verwendet werden, findet man bei StatCounter. Es ist auch möglich, die kompletten Daten als CSV zu erhalten.

So liegt z.B. der IE7 Ende 2012 unter 0,5%, IE8 liegt aktuell bei 7% (IE6 ist endlich bei 0,1%).

Webfonts

Durch Webfonts kann man nun beinahe beliebige Schriftarten auf Webseiten verwenden.

Dank Google ist die Einbindung sehr einfach (siehe head-Tag), es geht aber natürlich auch manuell.

Übungen

  1. Bauen Sie eine Header-Grafik ähnlich wie unter http://www.airport-nuernberg.de/ (ohne Animation). Es soll ein Hintergrundbild mit abgerundeten Ecken und einen davor liegenden Textbereich geben, der halb transparent ist. Die Schrift muss gut lesbar sein!
  2. Bauen Sie den Hintergrund von http://www.airport-nuernberg.de/ grob nach (ohne Wolken). Verwenden Sie einen CSS-Gradient, um auf dem Body-Tag einen Verlauf von blau (oben) nach weiß (unten) zu erstellen.
